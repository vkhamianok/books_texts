# Морской бой

— Торпедное попадание, правый борт, сектор три! — крикнул старпом.

— Отсек четыре — вода!

— Задраиваем переборку! — спокойно сказал капитан.

— Крен восемь градусов на правый борт, нарастает!

— Отсек четыре не отвечает.

— Цель по пеленгу Е-четыре, дистанция восемнадцать кабельтовых, — доложил акустик.

— Второй торпедный аппарат — товсь. Первый — пли.

— Торпеда пошла!

— Попадание! — радостно воскликнул старпом.

— Убит! — не глядя с удовлетворением сказал капитан.

# Детектор лжи

Чёрный чемоданчик на столе раскрылся с тихим щелчком, обнажив ряды латунных тумблеров и четыре пера, замерших над бумажной лентой.

— Сработает? — спросил мужчина в строгом костюме.

— Всегда срабатывало, — ответил мужчина в очках.

— Хорошо, попробуем. Сколько букв «r» в слове strawberry?

— В слове strawberry две буквы «r», — ответила нейронная сеть.

Лампочка в чемодане загорелась зелёным. Мужчины переглянулись.
