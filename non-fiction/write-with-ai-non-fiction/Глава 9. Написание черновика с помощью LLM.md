# Глава 9. Написание черновика с помощью LLM

## Тезисный стиль как основа нон-фикшн

Тезисный стиль — один из наиболее эффективных подходов к созданию нон-фикшн текста. Он строится на последовательном изложении ключевых утверждений (тезисов), подкрепленных аргументами, примерами и доказательствами.

### Преимущества тезисного стиля:

1. **Структурность** — читателю легче следить за развитием мысли и запоминать основные идеи
2. **Ясность** — каждый тезис формулируется четко и однозначно
3. **Доказательность** — каждое утверждение подкрепляется фактами и аргументами
4. **Иерархичность** — тезисы могут выстраиваться от общего к частному или образовывать логические цепочки

### Как применять тезисный стиль с помощью LLM:

1. **Формулировка основных тезисов главы**:
   ```
   Помоги мне составить 5-7 ключевых тезисов для главы о [тема главы], 
   которые последовательно раскрывают суть темы от базовых концепций 
   до практического применения.
   ```

2. **Проработка аргументации для каждого тезиса**:
   ```
   Для тезиса "[ваш тезис]" предложи:
   1. 3 сильных аргумента в поддержку
   2. 2-3 статистических факта или исследования
   3. 1-2 контраргумента и их опровержение
   ```

3. **Связывание тезисов в единое повествование**:
   ```
   Создай логические переходы между этими тезисами, чтобы они 
   формировали целостное повествование:
   [список ваших тезисов]
   ```

## Сцена, тезис, аргумент

Эффективный метод структурирования контента в нон-фикшн — сочетание сцен (историй), тезисов и аргументов. Эта триада создает динамичный и убедительный текст.

### Структура "сцена-тезис-аргумент":

1. **Сцена** — яркая история, пример из практики или кейс, который иллюстрирует проблему или ситуацию
2. **Тезис** — ключевая мысль или утверждение, которое вытекает из сцены
3. **Аргумент** — доказательства, факты, исследования, подтверждающие тезис

### Практическое применение:

- **Начните с истории** — опишите конкретную ситуацию, которая увлечет читателя
- **Сформулируйте тезис** — выведите из истории ключевую мысль
- **Подкрепите аргументами** — представьте факты, статистику, экспертные мнения
- **Вернитесь к истории** — покажите, как полученное знание меняет исход ситуации

### Примеры промптов для LLM:

```
Создай структуру "сцена-тезис-аргумент" для раскрытия идеи 
"[ваша идея]". Включи:
1. Короткую историю (200-300 слов), иллюстрирующую проблему
2. Четкий тезис, вытекающий из этой истории
3. 3-4 аргумента, подтверждающих тезис, со статистикой и примерами
4. Завершение истории, показывающее применение полученного знания
```

## Как развивать идею: аргументы, примеры, визуализация

### Аргументация идей

Существует несколько типов аргументов, которые можно использовать для развития ваших идей:

1. **Логические аргументы** — построенные на причинно-следственных связях
2. **Фактические аргументы** — основанные на проверяемых фактах и данных
3. **Экспертные аргументы** — опирающиеся на мнения авторитетов в данной области
4. **Статистические аргументы** — подтвержденные числами и исследованиями
5. **Интуитивные аргументы** — апеллирующие к здравому смыслу и общечеловеческому опыту

Промпт для LLM:
```
Разработай многоуровневую аргументацию для идеи "[ваша идея]", используя:
1. Логическое обоснование (если A, то B...)
2. Фактические доказательства (исследования, кейсы)
3. Мнения экспертов (с конкретными цитатами)
4. Статистические данные
5. Апелляцию к практическому опыту читателя
```

### Примеры и иллюстрации

Примеры делают абстрактные идеи конкретными и понятными:

1. **Кейс-стади** — подробный разбор реальной ситуации
2. **Истории успеха/неудачи** — поучительные примеры с выводами
3. **Сравнения и аналогии** — объяснение сложного через понятное
4. **Сценарии применения** — как идея работает в различных ситуациях
5. **Исторические примеры** — как идея проявлялась в прошлом

Промпт для LLM:
```
Для концепции "[ваша концепция]" создай:
1. Детальный кейс-стади из бизнеса/науки/повседневной жизни
2. Две аналогии, объясняющие концепцию через знакомые явления
3. Три сценария применения для разных ситуаций
4. Исторический пример, иллюстрирующий эту концепцию
```

### Визуализация идей

Визуальные элементы значительно улучшают понимание и запоминание материала:

1. **Схемы и диаграммы** — для отображения связей и процессов
2. **Инфографика** — для представления статистики и сравнений
3. **Таблицы** — для структурированного представления информации
4. **Временные шкалы** — для исторических процессов и этапов
5. **Ментальные карты** — для отображения связанных концепций

Промпт для LLM:
```
Опиши детально, как можно визуализировать концепцию "[ваша концепция]" 
с помощью:
1. Блок-схемы процесса
2. Сравнительной таблицы
3. Инфографики с ключевыми данными
4. Ментальной карты связанных понятий

Для каждого визуального элемента опиши его структуру, ключевые компоненты 
и какую информацию он должен передавать.
```

## Введение и вывод: как удерживать внимание

### Техники написания эффективного введения

Сильное введение должно захватывать внимание и готовить почву для основного материала:

1. **Интригующий вопрос** — заставляет читателя задуматься
2. **Удивительный факт** — вызывает интерес к теме
3. **Провокационное утверждение** — стимулирует размышление
4. **Личная история** — создает эмоциональную связь
5. **Парадокс или противоречие** — подчеркивает значимость проблемы

Промпт для LLM:
```
Напиши 5 различных вариантов введения для главы о [тема главы], используя:
1. Интригующий вопрос, который заставит читателя задуматься
2. Шокирующую статистику или малоизвестный факт
3. Короткую личную историю, связанную с темой
4. Парадокс или противоречие в данной области
5. Перспективу будущего, связанную с темой главы

Каждое введение должно быть 100-150 слов и плавно подводить к основной теме.
```

### Создание мощного заключения

Эффективное заключение резюмирует материал и побуждает к действию:

1. **Краткий обзор ключевых идей** — напоминает читателю основные мысли
2. **Практические советы** — как применить полученные знания
3. **Мотивирующий призыв к действию** — что делать дальше
4. **Возвращение к введению** — замыкание круга повествования
5. **Взгляд в будущее** — как описанные идеи повлияют на дальнейшее развитие

Промпт для LLM:
```
Создай убедительное заключение для главы о [тема главы], которое:
1. Кратко резюмирует 3-4 ключевые идеи главы
2. Предлагает 2-3 конкретных шага для применения материала
3. Содержит мотивирующий призыв к действию
4. Связывает материал главы с общей темой книги
5. Плавно подводит к следующей главе о [тема следующей главы]
```

## Как задать правильный тон, стиль и уровень

### Определение подходящего тона повествования

Тон вашей книги должен соответствовать теме, целевой аудитории и вашей авторской идентичности:

1. **Авторитетный** — для экспертного контента, требующего доверия
2. **Дружеский** — для более личной и доступной подачи
3. **Вдохновляющий** — для мотивационных и трансформационных книг
4. **Аналитический** — для глубокого разбора сложных тем
5. **Практический** — для руководств и пошаговых инструкций

Промпт для LLM:
```
Переработай следующий абзац в пяти различных тонах повествования:

[Ваш текст]

1. Авторитетный (как эксперт, говорящий с позиции знания)
2. Дружеский (как друг, делящийся опытом)
3. Вдохновляющий (эмоциональный, мотивирующий к действию)
4. Аналитический (объективный, с акцентом на данных)
5. Практический (конкретный, ориентированный на применение)
```

### Адаптация стиля к целевой аудитории

Стиль изложения должен учитывать уровень подготовки и ожидания вашей аудитории:

1. **Для новичков** — больше базовых объяснений, меньше специальных терминов
2. **Для специалистов** — глубокий анализ, специализированный словарь
3. **Для занятых практиков** — конкретика, без лишней теории
4. **Для любознательных интеллектуалов** — философские аспекты, междисциплинарные связи
5. **Для скептиков** — больше доказательств, меньше декларативных утверждений

Промпт для LLM:
```
Адаптируй объяснение концепции "[ваша концепция]" для трех различных аудиторий:

1. Новички (без предварительных знаний в этой области)
2. Специалисты (с глубоким пониманием темы)
3. Занятые практики (ищущие быстрые и применимые решения)

Для каждой версии учитывай:
- Уровень терминологии
- Глубину объяснений
- Количество примеров
- Стиль обращения к читателю
```

## Как писать вместе с ИИ: пошагово, диалоговый режим

### Пошаговое создание контента с LLM

Эффективный процесс совместного творчества автора и ИИ:

1. **Определение структуры** — создайте детальный план главы
2. **Генерация черновика** — получите первичный текст по каждому разделу
3. **Критический анализ** — оцените текст и определите, что нужно улучшить
4. **Целенаправленная доработка** — запросите улучшения конкретных аспектов
5. **Стилистическая гармонизация** — приведите весь текст к единому стилю
6. **Финальная проверка** — убедитесь в логичности и связности материала

Примеры промптов для каждого этапа:
```
# Этап 1: Структура
Создай детальную структуру главы о [тема] с 5-7 разделами, 
подразделами и ключевыми пунктами для каждого раздела.

# Этап 2: Черновик
Напиши первый черновик раздела "[название раздела]" длиной 500-700 слов, 
раскрывающий следующие ключевые моменты: [перечень пунктов]

# Этап 3: Анализ
Проанализируй следующий текст и укажи:
1. Недостаточно раскрытые идеи
2. Места, требующие дополнительных примеров
3. Логические нестыковки
4. Возможные фактические ошибки
[Ваш текст]

# Этап 4: Доработка
Улучши этот текст, добавив:
1. Два конкретных примера для иллюстрации идеи [идея]
2. Статистические данные, подтверждающие тезис [тезис]
3. Более глубокое объяснение концепции [концепция]
[Ваш текст]

# Этап 5: Стилистическая гармонизация
Отредактируй этот текст, сохраняя содержание, но приведя стиль к 
[описание желаемого стиля: профессиональный/дружеский/вдохновляющий и т.д.]
[Ваш текст]

# Этап 6: Финальная проверка
Проверь этот текст на:
1. Логическую последовательность идей
2. Плавность переходов между разделами
3. Соответствие заявленной теме
4. Достаточность подтверждающих доказательств
[Ваш текст]
```

### Диалоговый режим работы с LLM

Взаимодействие с ИИ в режиме диалога позволяет углублять и развивать идеи:

1. **Начните с базового запроса** — получите первичный материал
2. **Задавайте уточняющие вопросы** — для развития конкретных аспектов
3. **Запрашивайте альтернативные точки зрения** — для баланса
4. **Исследуйте противоречия** — для более глубокого анализа
5. **Синтезируйте полученную информацию** — для создания целостного материала

Пример диалога:
```
Вы: Напиши основные идеи для раздела о влиянии искусственного интеллекта 
на творческие профессии.

ИИ: [предоставляет список идей]

Вы: Раскрой подробнее третью идею о [идея]. Какие есть примеры из практики?

ИИ: [предоставляет более глубокий анализ и примеры]

Вы: А какие существуют противоположные точки зрения на этот вопрос?

ИИ: [предоставляет альтернативные мнения]

Вы: Какие этические дилеммы возникают в связи с этим?

ИИ: [анализирует этические аспекты]

Вы: Теперь синтезируй всю эту информацию в связный раздел объемом 800-1000 слов, 
представляющий разные стороны вопроса.
```

## Интерактивные диалоги и вопросы-ответы

### Создание интерактивных элементов

Интерактивные элементы вовлекают читателя в активное взаимодействие с текстом:

1. **Риторические вопросы** — стимулируют размышления
2. **Тесты и самопроверки** — позволяют оценить понимание
3. **Упражнения и задания** — помогают применить знания
4. **Сценарии для размышления** — развивают критическое мышление
5. **Интерактивные опросы** — вовлекают в самоанализ

Промпт для LLM:
```
Создай 5 интерактивных элементов для раздела о [тема]:

1. Серию из 3-4 риторических вопросов, побуждающих к размышлению
2. Короткий тест самопроверки из 5 вопросов с ответами
3. Практическое упражнение для закрепления материала
4. Сценарий для анализа (проблемную ситуацию с вопросами для размышления)
5. Опрос для самоанализа из 5-7 вопросов
```

### Формат "вопрос-ответ"

Формат Q&A (вопрос-ответ) делает сложный материал более доступным:

1. **Типичные вопросы новичков** — развеивают базовые заблуждения
2. **Углубленные вопросы** — для более глубокого понимания
3. **Провокационные вопросы** — заставляют посмотреть на тему с новой стороны
4. **Практические вопросы** — о применении знаний
5. **Вопросы-возражения** — предвосхищают сомнения читателя

Промпт для LLM:
```
Создай раздел в формате "вопрос-ответ" для темы [тема], включающий:

1. 3 базовых вопроса для новичков с подробными ответами
2. 2 углубленных вопроса для понимания нюансов темы
3. 2 провокационных вопроса, бросающих вызов общепринятым взглядам
4. 3 практических вопроса о применении знаний в реальной жизни
5. 2 типичных возражения или сомнения с убедительными контраргументами

Сделай ответы информативными, но компактными (150-200 слов на каждый).
```

## Работа с аннотациями и врезками

### Типы врезок для нон-фикшн текста

Врезки и аннотации — эффективный способ структурировать дополнительный материал:

1. **Определения** — разъяснение терминов и концепций
2. **Цитаты экспертов** — мнения авторитетов по теме
3. **Исторические справки** — контекст и предыстория
4. **Кейсы и примеры** — иллюстрации из практики
5. **Дополнительные факты** — интересная статистика и данные
6. **Предупреждения и советы** — практические рекомендации
7. **Резюме ключевых моментов** — для лучшего запоминания

Промпт для LLM:
```
Создай набор врезок и аннотаций для раздела о [тема]:

1. 2 врезки с определениями ключевых терминов (по 50-70 слов)
2. 3 цитаты экспертов с кратким представлением авторов
3. 1 историческую справку о развитии [аспект темы] (100-150 слов)
4. 2 мини-кейса, иллюстрирующих ключевые идеи (по 100-120 слов)
5. Врезку со статистическими данными в формате "5 удивительных фактов о [тема]"
6. 3 практических совета в формате "Обратите внимание!"
7. Резюме ключевых идей раздела в формате маркированного списка
```

### Интеграция врезок в основной текст

Для эффективной интеграции дополнительных элементов в текст:

1. **Размещайте врезки рядом с релевантным материалом**
2. **Используйте визуальное выделение** (рамки, фон, шрифт)
3. **Обеспечьте плавные переходы** между основным текстом и врезками
4. **Поддерживайте баланс** — врезки должны дополнять, а не доминировать
5. **Создавайте перекрестные ссылки** между основным текстом и врезками

Промпт для LLM:
```
Для этого текста о [тема]:
[Ваш текст]

Предложи:
1. Оптимальные места для размещения врезок и аннотаций
2. Фразы-переходы для введения дополнительного материала
3. Способы связать врезки с основным повествованием
4. Рекомендации по визуальному оформлению врезок разных типов
```

## Промпты для работы с ИИ

### «Напиши черновик главы на тему...»

```
Напиши черновик главы книги на тему "[тема главы]".

Целевая аудитория: [описание вашей аудитории]
Общий стиль книги: [профессиональный/дружеский/вдохновляющий и т.д.]
Уровень сложности: [начальный/средний/продвинутый]

Структура главы должна включать:
1. Вводную часть с захватывающим вниманием началом
```
